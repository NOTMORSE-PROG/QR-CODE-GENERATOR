# QR Code Generator

A simple, free, and instant QR code generator built with Next.js.

## Features
- Generate QR codes for any URL or text
- Download as PNG
- Clean, responsive design
- Free and open source

## Tech Stack
- Next.js
- React
- qrcode (npm package)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000)
